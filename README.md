# Aegis

**Identity-aware auth gateway in Go**  
JWT / JWKS validation • revocation • OIDC login • secure reverse proxy

---

## Purpose
Aegis is a lightweight Go gateway that provides JWT/JWKS validation and OIDC authentication in front of HTTP services.

## Features

- JWT validation using JWKS  
- Key rotation with caching  
- Route-based authorization  
- Secure reverse proxy  
- Header sanitation  
- OIDC login support
